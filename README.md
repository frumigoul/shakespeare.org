# shakespeare.org
A single, simple, but very large [emacs org-mode](https://orgmode.org) file of the [collected plays of Shakespeare](./shakespeare.org).

Buffer has:
  - 32,097 headings
  - 147,606 lines
  - 906,420 words
  - 5,358,505 characters

Useful for testing.
